PLAYER DE SAMPLES DO SPOTIFY

Player simples de samples do Spotify. 

Para usar, clique no botão "Dar Permissão de Acesso" para começar a procurar músicas pelo botão de busca. 

Caso queira adicionar a playlist, dê um duplo clique no item que deseja adicionar (em qualquer uma das células)

Caso queira salvar\carregar uma playlist, use as opções do menu.
